Blackwood

Your daily notes, from everywhere.

A local-first daily notes app that captures text, voice memos, photos, and handwritten notes — then makes all of it searchable through AI.

Download Getting Started

Features

Everything you need to capture and find your thoughts.

📝

Markdown Daily Notes

One document per day. Write, edit, auto-save.

🎤

Voice & Photos

Record audio or snap photos. Automatically transcribed and described by AI.

✍️

Viwoods Handwriting

Import handwritten notes from Viwoods AIPaper. OCR'd automatically.

💬

WhatsApp Integration

Send text, voice, or photos via WhatsApp. They appear in today's note.

✈️

Telegram Bot

Text, voice, and photos from Telegram. Uses long polling — no public URL needed.

🔍

RAG Chat

Ask questions about your notes. Get answers with source citations.

🔒

Local-First

SQLite database. Runs on your machine. Your data stays yours.

📄

PDF Export

Download any daily note as a clean, readable PDF.

📶

Offline Support

Works offline. Entries are queued and synced when the server is reachable.

🔐

HTTPS / TLS

Optional TLS with configurable cert and key paths.

⌨

Keyboard Shortcuts

Cmd+D today, Cmd+/ chat, Cmd+T timestamp, Cmd+Enter save.

How It Works

Up and running in under a minute.

1

Download the binary

Grab the latest release for your platform from GitHub.

2

Set your OpenAI API key

Export your key so Blackwood can transcribe, describe, and search.

3

Run and open

Start blackwood and open localhost:8080.

# Download the latest release curl -L https://github.com/csweichel/blackwood/releases/latest/download/blackwood_darwin_arm64.tar.gz | tar xz # Set your API key export OPENAI_API_KEY=sk-... # Start the server ./blackwood --data-dir ~/.blackwood

Architecture

A Go backend serving a React frontend over a single port.

┌─────────────────────────────────────────────────┐ │ Web UI (React) │ │ Calendar · Markdown Editor · Audio · Chat │ └──────────────────────┬──────────────────────────┘ │ Connect-RPC ┌──────────────────────┴──────────────────────────┐ │ Go API Server │ │ │ │ DailyNotesService · ChatService · ImportService│ │ WhatsApp Webhook · Telegram Bot │ │ Attachment Serving │ ├─────────────────────────────────────────────────┤ │ AI Pipelines │ │ Whisper (audio) · GPT-5.2 (vision/chat/OCR) │ │ text-embedding-3-small (semantic index) │ ├─────────────────────────────────────────────────┤ │ Storage │ │ SQLite (notes, entries, conversations) │ │ Embeddings (cosine similarity in Go) │ │ File system (attachments) │ └─────────────────────────────────────────────────┘

Installation

Download the pre-built binary for your platform.

# macOS (Apple Silicon) curl -L https://github.com/csweichel/blackwood/releases/latest/download/blackwood_darwin_arm64.tar.gz | tar xz # Run the server ./blackwood --data-dir ~/.blackwood
# macOS (Intel) curl -L https://github.com/csweichel/blackwood/releases/latest/download/blackwood_darwin_amd64.tar.gz | tar xz # Run the server ./blackwood --data-dir ~/.blackwood
# Linux (x86_64) curl -L https://github.com/csweichel/blackwood/releases/latest/download/blackwood_linux_amd64.tar.gz | tar xz # Run the server ./blackwood --data-dir ~/.blackwood
